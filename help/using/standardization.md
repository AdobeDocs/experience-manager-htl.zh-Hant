# 標準化 {#standardization}

以下是 HTL 標準：

* [HTL 規格](https://github.com/adobe/htl-spec) - 對語言專家來說，代表正式語法和功能說明。
* [HTL TCK](https://github.com/adobe/htl-tck) - 用來證明語言相容性的一套測試。
