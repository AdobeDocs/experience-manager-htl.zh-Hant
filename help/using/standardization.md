# 標準化 {#standardization}

以下是HTL標準：

* [HTL規格](https://github.com/Adobe-Marketing-Cloud/sightly-spec)\
   對於語言律師，請提供正式語法和功能說明。

* [HTL TCK](https://github.com/Adobe-Marketing-Cloud/sightly-tck)\
   一組測試，以驗證語言的相容性。
