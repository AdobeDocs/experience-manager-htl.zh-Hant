# HTL規格

HTML範本語言(HTL)先前稱為SiteC，已與Adobe Experience Manager6.0一起引入，並將JSP(JavaServer Pages)取代為HTML的偏好伺服器端範本系統。

此為「HTL規格」，定義語法和語言行為。

完整的HTL規格已詳細說明 [在Open Source Git存放庫中](https://github.com/adobe/htl-spec)
