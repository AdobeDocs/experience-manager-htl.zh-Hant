# HTL規格

HTML範本語言(HTL)先前稱為Sightly，現已隨Adobe Experience Manager 6.0推出，取代JSP(JavaServer Pages)，成為HTML偏好和建議的伺服器端範本系統。

這是HTL規格，可定義語言的語法和行為。

完整的HTL規格在我們的 [Open Source Git儲存庫中詳述](https://github.com/adobe/htl-spec)
